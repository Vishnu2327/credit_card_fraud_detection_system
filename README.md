# 💳 Credit Card Fraud Detection (EDA)

## 📌 Overview

This project performs **Exploratory Data Analysis (EDA)** on a credit card transactions dataset to understand the patterns behind fraudulent activities.

The focus is on uncovering insights, identifying anomalies, and preparing a strong foundation for building a fraud detection model.

---

## 🎯 Objectives

* Understand the structure of the dataset
* Detect data imbalance between fraud and non-fraud cases
* Analyze transaction behavior based on **amount** and **time**
* Visualize patterns that differentiate fraudulent transactions

---

## 📊 Key Highlights

* ⚠️ **Highly Imbalanced Dataset**
  Fraud cases are extremely rare compared to normal transactions

* 💸 **Transaction Amount Analysis**
  Fraudulent transactions often show different spending patterns

* ⏳ **Time-Based Patterns**
  Fraud occurrences vary across different hours of the day

* 📈 **Visual Insights**
  Graphs and plots help reveal hidden trends and anomalies
